# Showcase Muay Thai

Welcome to the official landing page repository for **Showcase Muay Thai**, Connecticut's newest fight promotion, presented by Campanella Martial Arts.

## About the Project

This project contains the single-page promotional website for Showcase Muay Thai. The site is designed to:
- Announce the arrival of a new, authentic Muay Thai promotion in Connecticut.
- Capture fan interest through a notification signup form.
- Provide information for fighters and gyms looking to compete in upcoming sanctioned events.
- Highlight the core values of the promotion: authenticity, fair matchmaking, and fighter safety.

## Tech Stack

The landing page is built entirely with fundamental web technologies for high performance and easy maintenance:
- **HTML5** for semantics and structure.
- **Vanilla CSS3** for styling, animations, and responsive design (no external frameworks like Bootstrap or Tailwind used).
- **Vanilla JavaScript** for simple interactive elements like the intersection observer for scroll animations.

## Key Features

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **Dynamic CSS Animations**: Features custom mesh drifts, pulse dots, and a continuous CSS-only ticker.
- **Intersection Observer Animations**: Elements subtly reveal themselves as the user scrolls down the page.
- **Modern Typography**: Utilizes Google Fonts (`Black Han Sans`, `Teko`, `Barlow`) to achieve a bold, fight-promotion aesthetic.

## Local Development

To view or edit the site locally:
1. Clone this repository to your local machine.
2. Open `index.html` in your web browser. No build steps or local server required!
